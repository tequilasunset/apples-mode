apples-mode
===========

An Emacs major mode for AppleScript.

System Requirements
-------------------

* GNU Emacs 23 or later
* AppleScript 2.0 or later

Features
--------

* Syntax highlighting
* Indentation
* Execution
* Highlight errors
* (De)compilation
* Completion
* Snippets
* Menu bar
